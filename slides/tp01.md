## TP setup

Dans le codespace, `$GOBIN` vaut `/go/bin`.
C'est là que l'exécutable de notre provider sera après un `go install .` depuis `/workspaces/go-tf-provider-lab/terraform-provider-json-file`.

Dans le codespace, le dossier `home` est `/home/vscode`.
C'est là qu'il faut ajouter un fichier `.terraformrc` avec la configuration suivante : 

```terraform
provider_installation {

  dev_overrides {
      "github.com/remieven/terraform-provider-json-file" = "/go/bin"
  }

  # For all other providers, install them directly from their origin provider
  # registries as normal. If you omit this, Terraform will _only_ use
  # the dev_overrides block, and so no other providers will be available.
  direct {}
}
```

Cela permet à Terraform de savoir qu'il doit utiliser le provider qu'on lui fournit en local.
